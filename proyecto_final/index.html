Proyecto final:
Gestión remota de maquina CNC

Objetivo:

Operar maquina CNC desde la interfaz del navegador por medio de una pagina WEB
desde donde se elijan los diferentes modos de operación y se puedan cargar archivos de operación
con los diseños ademas de contar con base de datos para registras los usuarios que deseen usar la maquina.
Para el desarrollo de este proyecto se contara con la maquina CNC que se encuentra en el laboratorio de
automatización y control.
